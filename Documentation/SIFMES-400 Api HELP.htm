SIFMES-400 Api HELP

Introduction

This is SIFMES-400 Api application. This API is used by some functionalities of SIFMES Delivery application and SIFMES-400 Web.

This API can be also used for third party applications.

MovService

APIDescription
GET MovService?option={option}

URI example: /api/movService?option=[Start/Stop/Restart]
Allows to start, stop or restart MovService application.
MovService has to be located on C:/MovService/SIFMES400.MovService.exe

Checker

APIDescription
GET Checker

URI example: /api/checker
Useful to know if the client has a connection to the API or not.

SignatureImage

APIDescription
GET SignatureImage?code={code}

URI example: /api/signatureImage?code=123456789012345
Allows to get the signature image made by the SIFMES Delivery application.

PerformanceAnalytics

APIDescription
GET PerformanceAnalytics?paOption={paOption}&from={from}&to={to}

URI example: /api/performanceAnalytics?paOption=AirAndEnergy&from=03/16/2023&to=03/30/2023
Allows to get the information displayed in the SIFMES-400 Performance Analytics dashboards.

Inventory

APIDescription
GET Inventory

URI example: /api/Inventory
Allows to get the inventory of SIF-400 system. The following information is included for each station:
SIF-401: Presence flag, minimum flag and container type for each container loader. Minimum flag, presence flag and loader position for pallet loader.
SIF-402: Hopper presence flag, minimum flag and color type for each hopper.
SIF-403: Minimum flag and maximum flag for water and oil tanks.
SIF-404: Minimum flag and container type for each line.
SIF-405: Count of caps and cap type for each feeder.
SIF-408: Count of packs.
SIF-410: Minimum flag of pack pallets.
SIF-412: Count of pack pallets inside each dock.

DeliveryNote

APIDescription
GET DeliveryNote?nfc={nfc}&rfid={rfid}

URI example: /api/deliveryNote?nfc=123456789012345
Allows to get the delivery note of a product delivered from station SIF-407 or SIF-412.

POST DeliveryNote

URI example: /api/deliveryNote
[POST param1: Option=NFC/RFID, param2: NFCId=1234567890123456, param3: RFIDId=1234567890ABCDEF, param4: Signature=base64Image]
Allows to send the image of the signature made for the delivery of a specific product.

WarehouseStatus

APIDescription
GET WarehouseStatus?stationTypeId={stationTypeId}

URI example: /api/warehouseStatus?stationTypeId=406
Allows to get all the information related to a warehouse station.

POST WarehouseStatus

URI example: /api/warehouseStatus
[POST param1: Location, param2: SKU]
Allows to add a new product inside the warehouse, or update a stored one.

DELETE WarehouseStatus

URI example: /api/warehouseStatus
[DELETE param1: SKUID=1]
Allows to delete a stored product from the warehouse.

Ekanban

APIDescription
GET Ekanban

URI example: /api/ekanban
Allows to get the number of remaining caps in station SIF-405.

DatabaseConnection

APIDescription
GET DatabaseConnection

URI example: /api/databaseConnection
Allows the API to refresh the database in use. This is useful when starting or finishing Educational Exercises.
